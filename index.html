Fast Car Rental

Sports Pro Technical Support App

The Sports Pro mobile app is a technical support portal tailored specifically for the needs of a hypothetical company named Sports Pro. The idea is that employees of Sports Pro can access and manipulate data about products, customers, incidents, and other staff through this easy to use app. Leveraging a MySQL database, the application features an array of essential features such as cookie integration, user authentication through login/logout functionalities, and a seamless search mechanism. Moreover, data manipulation is effortless, thanks to the PHP queries interfacing directly with the MySQL database. The aim is that the staff of Sports Pro will always have the required information at their fingertips, allowing for enhanced productivity and service excellence.

The site is live and you can view it here: Sports Pro Technical Support


Technologies used in this project:

  1. PHP
  2. MySQL
  3. HTML/ XML
  4. CSS
  5. JSON
  6. MAMP & Apache
  7. Made with Visual Studio

The Australian Census Questionnaire

In this project, I designed as a clone of the Australian Bureau of Statistics online form, utilising HTML, CSS, JavaScript, and JQuery. It features a structured form that enhances user experience through hidden help sections and input validation, ensuring ease of use and accuracy in data entry. The incorporation of a progress bar and collapsing sections not only streamlines the user journey but also contributes to the site's accessibility. What sets my site apart is its fully responsive design, which guarantees seamless functionality across all device screen sizes, making it a versatile tool for users on the go. The form is then able to be submitted by the user where it will be sent to a database and the user will be taken to a results page where they can see all of their entered data.


The site is live and you can view it here:
