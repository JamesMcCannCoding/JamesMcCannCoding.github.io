Fast Car Rental

Fast Car Rental is a groundbreaking venture based in Brisbane, offering car enthusiasts the unique opportunity to experience the thrill of driving iconic cars from the popular Fast & Furious franchise. This venture celebrates the fusion of cinematic automobile history with the thrill of real-world driving, all at the heart of Queensland. Although hypothetical, this innovative company boasts branches spread right across Queensland, ensuring accessibility for fans and thrill-seekers a-like. The Fast Car Rental website has been carefully crafted by myself using HTML, CSS, Bootstrap, and JavaScript, guaranteeing users a smooth, dynamic browsing experience along with animations and dynamic effects.

The site is live and you can view it here: www.FastCarRental.com.au

Sports Pro Technical Support App

The Sports Pro mobile app is a technical support portal tailored specifically for the needs of a hypothetical company named Sports Pro. The idea is that employees of Sports Pro can access and manipulate data about products, customers, incidents, and other staff through this easy to use app. Leveraging a MySQL database, the application features an array of essential features such as cookie integration, user authentication through login/logout functionalities, and a seamless search mechanism. Moreover, data manipulation is effortless, thanks to the PHP queries interfacing directly with the MySQL database. The aim is that the staff of Sports Pro will always have the required information at their fingertips, allowing for enhanced productivity and service excellence.

The site is live and you can view it here: Sports Pro Technical Support


Technologies used in this project:

  1. PHP
  2. MySQL
  3. HTML/ XML
  4. CSS
  5. JSON
  6. MAMP & Apache
  7. Made with Visual Studio
